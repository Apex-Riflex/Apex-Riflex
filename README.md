## 🎯 なぜGraphRAGか

前職のアニメCGプロダクション（株式会社サンジゲン）での開発経験から、制作現場の情報は「誰が・何を・誰と・どの話数で」という**リレーションの塊**だと気づきました。

そのリレーションを最もネイティブに扱えるのがグラフDB。そこにLLMを組み合わせて制作現場を支援する技術に、本気で取り組んでいます。
